# Fire Weather Index (FWI) Prediction System 🔥

## Project Overview
This project is an end-to-end Machine Learning web application built with Flask to predict the Fire Weather Index (FWI) and classify fire risks. It utilizes historical meteorological data to provide real-time risk assessments, helping to mitigate the impact of forest fires. 

## Dataset
The model is trained on the **Algerian Forest Fires dataset**, containing 240+ instances of weather observations. Key features used for prediction include:
* Temperature
* Relative Humidity (RH)
* Wind Speed (Ws)
* Rain

## Tech Stack
* **Programming Language:** Python
* **Machine Learning & Data Processing:** Scikit-learn, Pandas, NumPy
* **Web Framework:** Flask
* **Frontend:** HTML, CSS

## How to Run Locally
1. Clone the repository: 
   `git clone https://github.com/Harshit-Mishra123/fwi-prediction-system.git`
2. Install the required dependencies: 
   `pip install -r requirements.txt`
3. Run the Flask application: 
   `python app.py`
4. Open your web browser and navigate to `http://127.0.0.1:5000/`
